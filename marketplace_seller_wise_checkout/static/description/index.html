Odoo Marketplace Seller Wise Checkout

The module automatically separates and groups the products in the customer shopping cart according to the sellers. And allows the customers to place the separate orders for separate sellers from the marketplace. So now the customer’s orders can be split seller-wise for better product and order management.

Group Customer's Cart Seller Wise For Checkout On Odoo Marketplace

It splits customer's cart products seller-wise

  • As customers add the products to the cart, they are automatically grouped together seller-wise.
  • Makes it easier for the customer to see which products are from which seller from the cart at a glance.

Promotes separate independent checkout for each seller

Promotes separate independent checkout for each sellers.

  • If the customer prefers to buy products from one seller only, the cart shows a seller-wise checkout button.
  • After placing the order the products from other sellers are not lost so the customer need not build his cart again.

Detailed Features List

Split Order Seller-wise feature on Odoo marketplace

  • The module adds seller-wise cart features to Odoo Marketplace website.
  • Allows customers to sort and manage the cart according to sellers.

Automatically groups cart products according to sellers

  • The products added to cart by customer are automatically grouped seller-wise.
  • Each group has its own checkout button to place individual seller-wise orders.

Facilitates placement of seller-wise orders directly from Marketplace Cart

  • The customer can place the order for each seller separately.
  • Each seller-wise cart section needs to be placed as individual orders on Odoo marketplace.
  • These orders show up as separate orders in the Odoo backend.

Handle seller-wise orders from same customer separately in backend

  • Each order has its own delivery and invoice.
  • It makes seller-wise order management easier for the admin.

Cart Recovery after placing the order for one seller

  • The customer’s cart is auto-recovered after placing an order for one seller.
  • The remaining cart products are auto-added to the cart.
  • The customer can place the next order for another seller directly from the cart.

Display Accessories products for each seller separately in the Split cart

  • The module also splits accessory products for each seller in the cart.
  • Each seller-wise cart section shows the associated accessories products to the customer.
