Website Vouchers Vouchers

A promotional tool called “vouchers” are essentially provided by the sellers to increase sales, to become competitive, to re-activate old customers. Those customers that have been lured away by your competitors will start buying from you again when you give them a good reason to do so.

They feel special and privileged to get voucher codes for discounts to be applied on the cart. Voucher allows you to generate codes, which can be used to provide flat or percentage based discount on a customers order. It will be applicable on the store front checkout. Additional filtering can be applied so that customer uses voucher for single time use spends certain amount for a specific time period to access the voucher code.

Salient Features
  • Create Vouchers/Coupons for discount on special occasions for grabbing more customers.
  • Create vouchers in simple steps and set some default configurations for saving time.
  • Define certain rules for the voucher applicability.
  • Same voucher can be used for Website, Odoo Core or POS .
  • Create vouchers for Special Customers or for All customers.
  • Create vouchers for Specific Products or All Products.
  • Display vouchers to customers in Website.
  • Allow your customers to partially redeem vouchers and set maximum partial redeem limit.
  • A unique code is generated for every voucher.
  • Generate Voucher code automatically or manually.
  • Notify customers via email for which you are creating vouchers.
  • Maintain the history of vouchers used like order in which and who used etc
  • Maintain the Credit and Debit of Vouchers using history.

Configuring and creating Odoo Vouchers

Default configuration of vouchers

Some key points

  • In order to save time for creating vouchers set some default configurations.
  • The default values will be used while creating the vouchers.
  • Vouchers can not be created below and above the minimum and maximum value respectively.
  • Vouchers can be created for some Specific customer or for All customers
  • Same voucher can be used for the Ecommerce & Odoo Core or POS or For Both.
  • Depending upon the customer type selected we can partially use the voucher and also specify the maximum limit of the partial use.
  • Vouchers can be created for All products or for some specific products.
  • Vouchers can be applied for some minimum cart value only.
  • Value (-1) represents unlimited wherever used.

Creating Vouchers

Some key points

  • Create vouchers/coupons in easy steps. Fetch most information from default configuration.
  • Code of voucher is auto generated(if left blank) and manual.
  • Validity can be applied by using days and expiry date.
  • Vouchers created are applicable only after the Applicable date.
  • Vouchers can be created for some Specific customer or for All customers
  • Same voucher can be used for the Ecommerce & Odoo Core or POS or For Both.
  • Creating vouchers that could be applicable to Specific Products or All Products.
  • Depending upon the customer type selected we can partially use the voucher, and also specify the maximum limit of the partial use.
  • Vouchers can be also applied based on the amount in your Shopping Cart or POS order dashboard. If the order is less than a particular amount then voucher can not be applied.
  • Notify your customers via email for which you are creating vouchers/coupons.

History and Reciept view

Some key points

  • Keep the record of the vouchers used with order number, customer etc.
  • Print also the vouchers you are creating.
